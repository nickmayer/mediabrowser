About Media Browser

Formerly known as Video Browser, Media Browser is a personal media aggregator that takes your recorded, digital, or ripped media and presents it in a simple, easy to use interface. Media Browser prides itself on delivering a fast, intuitive, media rich experience for the whole family.

 

Setup Media Browser

Learn how to setup Media Browser by visiting our wiki. You'll find information on best practises to organize your media for maximum compatibility.

 

Technical Information

Media Browser is written in the C# programming language targetted for the .NET 3.5 SP1 Framework. You can download the .NET Framework from Microsoft or directly form here. Media Browser runs on the Microsoft Media Center platform, found on Vista Home Premium, and Vista Ultimate operating systems.

The UI for Media Browser is written in Windows Media Center Markup Language (MCML), which is an XML-based declarative language. MCML rendering technology allows Media Browser to create experiences with the same richness and quality of the user interface (UI) within Windows Media Center (perhaps even more so).

Media Browser beta v2.0.10.0 and above may also run on Microsoft Windows 7, however, please note that due to Windows 7's beta status no guarantees are made by the Media Browser team that it will work for you. You can interact with others using Media Browser on Windows 7 on our forum here.

 

Free Open Source Software

Media Browser is an open source software project licensed under the GNU General Public License. The Media Browser project is a non-profit project run and developed by volunteers in their spare-time without any monetary gain.
