How do I add my video folders to VB?

VB expects all your video files to be in your user account's "My Videos" directory. If you do not want to keep your files there, the easiest way is to create a link to your directory.

Alternatives
  • Virtual folders, see How do I get VB to show the contents of multiple directories?
  • Set your accounts video directory to a different path. You can do that manually by using regedit and modify the following key "HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders\My Video" (if it doesn't exist yet, create it with the type REG_EXPAND_SZ). If you don't want to do that manually, SMO has a GUI for it located in the "extras" menu.

How do I get VB to show the contents of multiple directories?

VB supports virtual folders. Create a new textfile on your videos directory with the ending .vf (not .vf.txt!), for example: Movies.vf

Inside this textfile, add one line for each directory you want to add using the folder: statement: 

folder: d:\movies\
folder: e:\other movies\

When you start VB, the file Movies.vf will show up like a regular directory. If you open it, you will see all the movies that are present in the two specified locations.

To get VB to display an folder icon, you'll have to tell it manually using the image: statement: 

image: d:\movies\folder.jpg
folder: d:\movies\
folder: e:\other movies\

Why doesn't VB allow me to select individual video files in a directory?

VB needs to have a separate directory for each movie. In some cases one movie might be split in two or more files, therefore VB automatically creates a playlist and plays all the files in a directory in sequence. If the files really are separate movies, you will have to put them in separate directories.

Examples:

  • d:\movies\First Movie (2007)\
    • movie.avi
  • d:\movies\Second Movie (2008)\
    • bm-cd1.avi
    • bm-cd2.avi

In the case of tv shows, the files need to be grouped in a season folder inside the shows main folder:

  • d:\TV Shows\Great Show\Season 1\
    • 101 - Episode 1.avi
    • 102 - Episode 2.avi
  • d:\TV Shows\Great Show\Season 2\
    • 201 - Episode 1.avi
    • 202 - Episode 2.avi

See Where do I get metadata for TV shows? for getting metadata and naming your files correctly.

If you really want to disable the the playlist feature, put a textfile in the affected folder with the content noautoplaylist

The official way would be the original ruby script, see this page for detailed instructions. A GUI driven alternative is "TV Scout", available in the VB forum.

Where do I get metadata for movies?

There are several tools available to get metadata for movies. VB was built to use the XML files created by MyMovies's collection manager. Since this is only a small part of a huge software package, other solutions that are more lightweight and tailored specifically for VB are also available:

  • MetaBabel (Source: DVD Empire)
  • salami's movie organizer (Sources: IMDb.com, impawards.com, moviemeter.nl)

How do I get VB to show images for the genre folders?

VB is currently lacking good default genre images. If you have some artistic skills, please show us your work in the forum!

To add the Images manually, you have to place them in the Directory: C:\Users\<mce user name>\AppData\Roaming\SamsVideoBrowser\Genres

Name those files <genre>.jpg, for example Action.jpg

How can I manually force VB to update it's cache?

VB uses a local cache to speed things up. This feature is very new and might not work perfectly yet, if your VB suddenly starts to stop displaying new information, simply delete all the contents in the folder: C:\Users\<username>\AppData\Roaming\SamsVideoBrowser (Cache has been moved to c:\ProgramData\VideoBrowser in v. 1.1.12.0 where the new config file is also located)
