Configuration

There are two types of configuration in Video Browser. The first is within Video Browser itself, and to demonstrate we have a simple video cycling through the different presentation options.



The second type of configuration is more advanced and requires editing a configuration file in a text editor such as Notepad.
This configuration file is an XML file, so with any changes you make, it is important that you do not "break" the way the values are closed. If this sounds a bit too daunting, you can always delete the configuration file to go back to the default settings.

The configuration file is found in C:\ProgramData\VideoBrowser\VideoBrowser.config. Most values you will want to leave as the defaults, but several values are of interest

VideoBrowser.Config

Option Default Value Description
InitialFolder MyVideos Defaults to the Window's users "My Videos" folder. This can be set to any accessible folder, including network shares.
ie, <InitialFolder>\\myserver</InitialFolder>
ListViewWatchedColor LightSkyBlue Once you've watched a video in VideoBrowser, it will be coloured differently so you know you've watched it. This is just a standard .NET colour, it can be set to values like Red, White, Black, Purple, Gray, etc
EnableUpdates true While it's recommended to always have the latest version, often you don't want want the prompt popping up. Setting this to false will stop it from checking/asking.
DefaultViewType Detail Can be set to Detail, Poster or Thumb.
